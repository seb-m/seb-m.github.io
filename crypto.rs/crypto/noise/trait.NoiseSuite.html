fn encrypt(&self, key: &mut [u8], nonce: &mut [u8], plaintext: &[u8], pad_len: Option<uint>, authtext: Option<&[u8]>) -> Result<SBuf<A, u8>, ()>

Encrypt plaintext, authenticate its result along with authtext. Update its context key and nonce and eventually return the ciphertext. When None is returned key and nonce are not updated. pad_len should be None when no padding should be appended.

fn decrypt(&self, key: &mut [u8], nonce: &mut [u8], ciphertext: &[u8], pad: bool, authtext: Option<&[u8]>) -> Result<SBuf<A, u8>, ()>

Authenticate authtext and the encrypted data cyphertext, then decrypt ciphertext, update key and nonce and eventually return its plaintext. When None is returned key and nonce are not updated. pad is true when some padding was appended to the plaintext and must be removed, the plaintext is returned without padding. If the authentication failed None is returned.

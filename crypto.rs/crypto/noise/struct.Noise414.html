Noise414 cipher suite

Use the following primitives:

  • Curve41417 (DH)
  • Chacha20-Poly1305 (Authenticated Encryption)
  • Shake256 (Key-Derivation)

Note: this cipher suite is different than the suites officially supported. It uses a DH function with a different primitive, the same ENCRYPT function than the one specified in Noise255 and a different KDF function with a different primitive.

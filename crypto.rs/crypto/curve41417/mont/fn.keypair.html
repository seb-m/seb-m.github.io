pub fn keypair<A: Allocator = GuardedHeapAllocator>() -> (MontPoint<A>, Scalar<A>)

Generate a new key pair

A new key pair (pk, sk) is generated. sk is a secret key randomly generated and used as scalar value to compute the public key pk where pk=sk.BP with BP as base point. sk is clamped (see B416::clamp_41417()). The scalar value is returned wrapped in Scalar and the public key is wrapped in MontPoint.

Required Methods

fn encrypt<T: Encrypt<S>>(&mut self, value: &T, output: &mut [u8]) -> IoResult<uint>

Encrypt the encryptable value and put the result in output. The size of output must at least be of the size of input. Return the number of encrypted bytes written to output.

Provided Methods

fn decrypt<T: Encrypt<S>>(&mut self, value: &T, output: &mut [u8]) -> IoResult<uint>

Decrypt value in output. See encrypt operation.
